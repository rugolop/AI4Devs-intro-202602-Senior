# PROMPT
```Vamos a crear una pequeña herramienta para transformar una cadena de texto. Construye una pagina web en html (index.html) y pon la logica en un archivo javascript (script.js). La pagina web tendrá un elemento input para introducir la cadena de texto, un botón que ejecutará la lógica y una etiqueta para mostrar el resultado. La funcionalidad principal, se trata de devolver el texto al revés, es decir, si se introduce el texto Hola se deberá devolver aloH, se deben respetar los caracteres en mayuscula y en minuscula, por ejemplo: LaBeRinto, devolverá: otniReBaL. El botón deberá ser visible sólo si el input tiene más de tres caracteres.```
